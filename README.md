# no-slop

TUI/CLI based Agentic AI client with minimal vibe-coding.

## Why?

I want to just enjoy writing code again. Very, very slowly. It will be a nice change of pace.

I start out with very to little AI, then I use the `no-slop` client to bootstrap my way back into the dark, dark land of vibe coding. Hah. Fun.

I also bought a bunch of mystery-box clearance/refurbed mini PCs from GMKTec and want to just create an "OpenClaw" to do random stuff.

## How?

- A desktop environment with very minimal resources so I am not tempted to go "AI".
- Linux Mint, VSCodeium (the OSS version with no Copilot stuff), Helium browser
- Minimal AI: LMStudio with a local LLM to just help me with grammar checking, writing and of course, testing the agent we are actually making.
- Documentation will searched the old school internet way and read the painful, slow way.

## When?

It's a side project. Whenever I get bored of what I am doing.

## What?

I want to have an AI agent just do stuff.
